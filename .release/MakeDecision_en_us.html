Make decision: the method description

ƒ RD AAOW FDL


The Make decision app illustrates a non-linear method for analyzing hierarchies. The method involves comparing some objects, phenomena or outcomes of events based on a set of criteria and the choice of the most “correct” of them, i.e. that one, which will have the highest score. The user is prompted to select elements and criteria for their comparison, as well as to evaluate the significance of each criterion and the “cost” of the element for each criterion. Based on this information, the program receives a result.


Controls (for Windows):

  • Enter – enter values to lists
  • Shift + Enter – going to the next step
  • Home – restarting the sequence
  • F1 – opening the App about window
